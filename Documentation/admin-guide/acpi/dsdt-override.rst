===============
Overriding DSDT
===============

LinaOS supports a method of overriding the BIOS DSDT:

CONFIG_ACPI_CUSTOM_DSDT - builds the image into the kernel.

When to use this method is described in detail on the
LinaOS/ACPI home page:
https://01.org/linaos-acpi/documentation/overriding-dsdt
